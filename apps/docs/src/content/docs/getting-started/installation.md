---
title: Installation
description: Install Minions Oss-content packages
---

## TypeScript / Node.js

```bash
npm install @minions-oss-content/sdk minions-sdk
```

## Python

```bash
pip install minions-oss-content
```

## CLI

```bash
npm install -g @minions-oss-content/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
